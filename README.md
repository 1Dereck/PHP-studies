## 📌 Sobre o repositório

Este repositório contém meus estudos e projetos desenvolvidos durante minha jornada de aprendizado em **PHP**. O objetivo é evoluir desde os fundamentos da linguagem até conceitos avançados de desenvolvimento backend, arquitetura de software e sistemas escaláveis.

Os estudos seguem trilhas da **Alura** combinadas com prática em projetos reais.
